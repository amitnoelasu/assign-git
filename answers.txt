• What is a gitignore file and what should it include for a Java project?
It is file used to ignore certain types of files that may include senitive information or machine-specific information
It should include the following files:
# Compiled class file
*.class

# Log file
*.log

# BlueJ files
*.ctxt

# Mobile Tools for Java (J2ME)
.mtj.tmp/

# Package Files #
*.jar
*.war
*.nar
*.ear
*.zip
*.tar.gz
*.rar

# virtual machine crash logs, see http://www.java.com/en/download/help/error_hotspot.xml
hs_err_pid*


• What does fast-forward in git mean?
Fast forward means to update the HEAD pointer in such a way that its new value is a direct descendant of the prior value.
